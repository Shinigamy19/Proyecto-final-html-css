# Código Dein — Landing Page

Bienvenido al repositorio oficial de la **Landing Page** para **Código Dein**, una marca de ropa con estilo único y enfoque moderno.

## 🚀 Descripción

Código Dein es una marca ficticia de ropa con identidad propia, pensada para quienes buscan destacar su estilo con piezas únicas y modernas.
Esta landing page está diseñada para presentar la marca **Código Dein** al mundo, capturar la atención de los visitantes y facilitar la comunicación con clientes potenciales. 
Está construida con **Tailwind CSS** para un diseño ágil y responsivo, y cuenta con componentes interactivos en **JavaScript** para una experiencia de usuario atractiva.

---

## ✨ Características principales

- ✅ **Diseño responsivo:** Adaptable a todos los tamaños de pantalla.
- 🌙 **Modo oscuro:** Cambio dinámico entre modo claro y modo noche para mayor comodidad.
- 🎞️ **Slider interactivo:** Muestra colecciones o productos destacados.
- 📨 **Formulario de contacto:** Permite recibir mensajes y consultas de los usuarios.
- ⚡ **Componentes JS reutilizables:** Scripts para toggle de tema, slider, validación del formulario, entre otros.

---

## 🛠️ Tecnologías utilizadas

- **HTML5**
- **Tailwind CSS**
- **JavaScript Vanilla**
